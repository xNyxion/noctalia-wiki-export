Manage your hardware and system state directly through the shell.

### Audio Management

| Function | Command | Description |
|----------|---------|-------------|
| **Volume Up** | <IpcCommand command="volume increase" /> | Increase output volume |
| **Volume Down** | <IpcCommand command="volume decrease" /> | Decrease output volume |
| **Mute Output** | <IpcCommand command="volume muteOutput" /> | Toggle output mute |
| **Input Up** | <IpcCommand command="volume increaseInput" /> | Increase input volume |
| **Input Down** | <IpcCommand command="volume decreaseInput" /> | Decrease input volume |
| **Mute Input** | <IpcCommand command="volume muteInput" /> | Toggle input mute |
| **Toggle Panel** | <IpcCommand command="volume togglePanel" /> | Toggle audio panel |
| **Open Panel** | <IpcCommand command="volume openPanel" /> | Open audio panel |
| **Close Panel** | <IpcCommand command="volume closePanel" /> | Close audio panel |

### Media Controls

| Function | Command | Description |
|---|---|---|
| **Play/Pause** | <IpcCommand command="media playPause" /> | Toggle playback |
| **Play** | <IpcCommand command="media play" /> | Play media |
| **Pause** | <IpcCommand command="media pause" /> | Pause media |
| **Next** | <IpcCommand command="media next" /> | Next track |
| **Previous** | <IpcCommand command="media previous" /> | Previous track |
| **Seek Rel.** | <IpcCommand command="media seekRelative $offset" /> | Seek by seconds |
| **Seek By Ratio** | <IpcCommand command="media seekByRatio $position" /> | Seek to position (0.0-1.0) |
| **Toggle Panel** | <IpcCommand command="media toggle" /> | Toggle media panel |

### Display & Brightness

| Function | Command | Description |
|----------|---------|-------------|
| **Brightness Up** | <IpcCommand command="brightness increase" /> | Increase brightness |
| **Brightness Down** | <IpcCommand command="brightness decrease" /> | Decrease brightness |
| **Set Brightness** | <IpcCommand command="brightness set <value>" /> | Set brightness (0-100) |

### Battery Management

| Function | Command | Description |
|----------|---------|-------------|
| **Charging Modes** | <IpcCommand command="batteryManager cycle" /> | Cycle charging modes |
| **Full Capacity** | <IpcCommand command="batteryManager set full" /> | Set Full capacity mode |
| **Balanced** | <IpcCommand command="batteryManager set balanced" /> | Set Balanced mode |
| **Lifespan** | <IpcCommand command="batteryManager set lifespan" /> | Set Lifespan mode |
| **Battery Panel** | <IpcCommand command="battery togglePanel" /> | Toggle battery panel |

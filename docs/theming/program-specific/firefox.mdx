Firefox can be themed using the Pywalfox extension, which syncs Noctalia's color scheme to your browser.

## Setup

1. **Install Pywalfox:**
   ```bash
   # Arch Linux / Manjaro
   paru -S python-pywalfox
   ```

2. **Install Pywalfox browser extension:**
   - Install the [Pywalfox extension](https://addons.mozilla.org/en-US/firefox/addon/pywalfox/) from Firefox Add-ons

3. **Enable Firefox theming in Noctalia:**
   - Open **Settings → Color Scheme → Templates → Programs**
   - Toggle on **Firefox (Pywalfox)**

## Troubleshooting

- **Extension not connecting:** Check that the browser extension is installed and enabled

## Additional Configuration

Pywalfox will automatically sync your Noctalia color scheme to Firefox. The theme updates in real-time when you change color schemes in Noctalia, as long as the daemon is running.
